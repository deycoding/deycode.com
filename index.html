Independent Research - Built and Launched

These are the projects I have built from ground up - each designed to solve a real-world problem. During these projects, I have trained multiple foundation models spanning Dense decoders, Mixture of Experts, and Encoder architectures.

  • Compliance Classifier Router (134M Encoder)
    A BERT-based encoder model that classifies incoming queries by complexity and PII sensitivity, routing them to the appropriate LLM tier. 99.2% accuracy, 7ms latency, deployed on ECS with full CloudFormation IaC. Enforces data residency for Indian financial services.
  • Indian Multi-Language Model (3B)
    Pre-trained on 6 Indian languages - Hindi, Bengali, Tamil, Telugu, Marathi, Kannada. Uses a 64K multilingual tokenizer trained across all languages. Designed for a country where most AI tools only work in English.
  • Sanskrit Translation Model (3B-7B)
    Bilingual English-Sanskrit model trained on Vedic texts, epics, and modern Sanskrit. Handles translation, grammar correction, and transliteration. Preserving an ancient language through modern architecture.
  • Drone Programming - Instruction Set Generator (<1B)
    A decoder model that converts English prompts into executable drone flight code. Non-programmers describe a mission and the model generates the SDK commands. Includes a safety validator that checks altitude limits and whitelisted functions before execution.
  • Indian SEBI Regulation - Guideline Decoder (1B)
    A decoder model trained on SEBI circulars, regulations, and securities laws. Compliance teams ask questions in natural language and the model generates answers citing specific regulation sections. Purpose-built for Indian financial services.
